# Tech News Scraper

A simple Python project that fetches the top 5 tech headlines from TechCrunch using BeautifulSoup.

##  Requirements

- Python 3.x
- `requests` module
- `beautifulsoup4` module

##  How to Run

1. Install dependencies:
pip install requests beautifulsoup4

2. Run the script:
python scraper.py

##  Tools Used

- Python
- Requests
- BeautifulSoup
